x1 - liczba dłuznic cietych I sposobem
x2 - liczba dłużnic cietych II sposobem

300 kompletow w kazdym 7 belek o dlugości 0,7 m i 4 belki o dlugości 2,5m

Y = min(0,3x1+0,2x2)
x1 >= 0
x2 >= 0

7x1 + 3x2 >= 2100
2x2 >= 1200
